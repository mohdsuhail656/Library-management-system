# Library Management System

This is a simple Library Management System I built using C++ and File Handling as part of my learning journey.

The idea was to create something useful while practicing the concepts I was studying — like file handling, OOP, and 
basic data management — without relying on any database.

## What it does

- Add a new book
- Issue a book to a student
- Return a book
- Search for a book
- Delete a book record

All data is saved to a file, so nothing is lost when you close the program.

## How to run it

You need a C++ compiler like g++.

```bash
g++ main.cpp -o library
./library
```

That's it. No extra setup needed.

---

## Why I built this

I wanted to practice File Handling in C++ and understand how data can be stored without using a database.
This project helped me understand fstream, how records are written and read from files, and how to structure a small CLI application.

---

## Things I learned

- File Handling (fstream, ifstream, ofstream)
- Structuring a project with classes
- How to handle user input properly in a CLI app
